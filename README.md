# Scoreboard-OBS
Pool Scoreboard for WIT Billiards for OBS Overlay

# Install

- `git clone https://github.com/jamesql/scoreboard.git`
- `cd ./scoreboard`
- `npm i`
- `npm start`

# Add to OBS
- Add new browser
- Set url to `http://localhost:3000/`
