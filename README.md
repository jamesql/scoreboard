# Scoreboard-OBS
Scoreboard overlay for OBS that can be used for Pool, E-Sports, Soccer, Hockey and more!

# Install

- `git clone https://github.com/jamesql/scoreboard.git`
- `cd ./scoreboard`
- `npm i`
- `npm start`

# Add to OBS
- Add new browser
- Set url to `http://localhost:3000/`
